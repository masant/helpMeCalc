Some Tips

  1. Enter with the Numerator from your System Transfer Function

    2. You have to put all coefficients from the numerator. For example, if the numerator has a polynom like 

    2x³ + 3x² + 6x + 12,
    you have to put in the input 'Numerator' as the following: 
    2 3 6 12
    .

  2. Complete the Denominator from your System Transfer Function

    3. You have to follow the same logic used to enter the numerator, explained above.

  3. Press the "Run" button

    4. After you complete all the inputs, you just need to press the 'run' buttom and that's all! Easy, right? :)

    Some Theory

    In control theory and stability theory, root locus analysis is a graphical method for examining how the roots of a system change with variation of a certain system parameter, commonly a gain within a feedback system.

    This is a technique used as a stability criterion in the field of classical control theory developed by Walter R. Evans which can determine stability of the system.

    The root locus plots the poles of the closed loop transfer function in the complex s-plane as a function of a gain parameter (see pole–zero plot).

    A graphical method that uses a special protractor called a "Spirule" was once used to determine angles and draw the root loci
